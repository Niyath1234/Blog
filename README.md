# Blog

**The Multiverse of Intelligence** · Neo-brutalist, graphic-novel style research blog (Next.js 16, Tailwind, React Three Fiber).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy on [Vercel](https://vercel.com/new): import this repo, framework preset **Next.js**, root directory **.** (repo root if this folder is the repo).

See [Next.js deployment docs](https://nextjs.org/docs/app/building-your-application/deploying).
